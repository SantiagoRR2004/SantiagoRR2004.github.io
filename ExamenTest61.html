1: Which of the following correctly describe the differences between **structured** and **unstructured** data? [Tema 6Teacher]

  • A) Structured data requires cognitive computing for interpretation.
  • B) Structured data cannot be used for business decisions.
  • C) Unstructured data often requires machine learning for analysis.
  • D) Unstructured data includes text, video, and images.
  • E) Structured data is easier to store and query using RDBMS.

Respuestas correctas: C, D, E

2: Which of the following most accurately characterizes the role of APIs in cloud-based IoT platforms? [Tema 7Teacher]

  • A) APIs are used to hard-code device behaviors in firmware.
  • B) APIs facilitate interoperability and programmatic interaction among services, devices, and data.
  • C) APIs replace the need for middleware in IoT applications.
  • D) APIs enable human operators to configure cloud dashboards manually.

Respuestas correctas: B

3: An automotive manufacturer wants to implement predictive maintenance using vehicle sensor data globally. Which architecture is best? [Tema 7Teacher]

  • A) Edge-only model storing data locally in each vehicle.
  • B) Cloud-based infrastructure using IaaS for scalable compute and storage.
  • C) Manual data transfer to offline data centers for model training.
