1: Which of the following statements best explains the relationship between MapReduce and fog computing? [Tema 7Teacher]

  • A) Fog computing applies MapReduce principles to localize processing and reduce data transfer overhead.
  • B) Fog computing only supports the reduce phase of MapReduce to simplify architecture.
  • C) Fog nodes use HDFS to cache raw sensor data for map operations.
