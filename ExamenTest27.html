1: Which of the following are benefits of using OpenStack as a cloud reference architecture for IoT systems? [Tema 7Teacher]

  • A) RabbitMQ enables tightly coupled communication between cloud nodes.
  • B) Modular components like Nova and Glance allow for flexible resource control.
