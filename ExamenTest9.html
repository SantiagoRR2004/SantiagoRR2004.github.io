1: Which of the following challenges must IoT-cloud platforms address to ensure interoperability and effective integration? [Tema 7Teacher]

  • A) Data quality and timestamp inconsistencies.
  • B) Fixed data schemas enforced by IoT gateways.
  • C) Semantically diverse data formats.
  • D) Homogeneous reference units across all devices.
  • E) Presence of virtual and physical sensors.

Respuestas correctas: A, C, E

2: What distinguishes a Raspberry Pi from an Arduino? [Tema 5Teacher]

  • A) Arduino supports high-definition video output.
  • B) Raspberry Pi has analog input support built-in.
  • C) Arduino has a faster processor than Raspberry Pi.
  • D) Raspberry Pi runs a full OS and supports high-level languages.

Respuestas correctas: D
