1: Which of the following correctly describe the differences between **structured** and **unstructured** data? [Tema 6Teacher]

  • A) Unstructured data includes text, video, and images.
  • B) Unstructured data often requires machine learning for analysis.
  • C) Structured data is easier to store and query using RDBMS.
  • D) Structured data cannot be used for business decisions.
  • E) Structured data requires cognitive computing for interpretation.

Respuestas correctas: A, B, C
