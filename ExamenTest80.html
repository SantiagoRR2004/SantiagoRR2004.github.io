1: An automotive manufacturer wants to implement predictive maintenance using vehicle sensor data globally. Which architecture is best? [Tema 7Teacher]

  • A) Manual data transfer to offline data centers for model training.
  • B) Fog nodes in dealerships processing data and sending logs quarterly.
  • C) Cloud-based infrastructure using IaaS for scalable compute and storage.
  • D) Edge-only model storing data locally in each vehicle.

Respuestas correctas: C

2: Which of the following business motivations drive the convergence of IoT and cloud computing? [Tema 7Teacher]

  • A) The need for device-specific control interfaces.
  • B) Support for computationally intensive AI services.
  • C) Avoidance of CAPEX in early IoT start-ups.
